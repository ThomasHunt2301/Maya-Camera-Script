How to install Camera Script:

Instructions:

1)Install Maya_Script_Fin from github
2)Located script directory (located in User Drive\Autodesk\Maya(version)\scripts
3)Put Maya_Script_Fin.py into directory
4)In maya open Script editor (Bottom Right of the UI)
5)Then File > Open Script then load newly place script.
6) You can run script using the Execute all button (double play button)
7)alternatively you can also File > source script and then will run the code straight away



Thank you!
